- 커서 드래곤 -
 버전 1 기준 현재 일반, 백그라운드 작업, 불가능, 도움말 커서가 작업되어
있어요!

cursor_def.ani - 일반 선택 커서입니다.
cursor_help.ani - 도움말 선택 커서입니다.
cursor_no.ani - 선택 불가 커서입니다.
cursor_bkg.ani - 백그라운드 작업 커서입니다.

각 커서에 맞도록 [마우스-포인터 설정]에서 지정해주세요!
추후 모든 커서가 작업이 끝나면 자동설정 파일 (inf)도 추가해놓을게요.


- 시작메뉴 여우 -
Windows 7의 경우 Start Orb Changer를, 8,8.1,10의 경우 Classic Menu 또는
Start10 등 시작메뉴를 변경해주는 프로그램을 통해 등록하셔야 합니다.

필자는 Classic Menu를 사용 중입니다. Behaivor에서 클릭 시 원래 시작 메뉴가
나오도록 설정하면 평소처럼 사용할 수 있습니다!


- 휴지통 드래곤 - 
[바탕 화면 아이콘 설정] 에서 아래와 같이 지정해주세요.

rcbin_fll.ico - 휴지통(가득 참)
rcbin_def.ico - 휴지통(비어 있음)


- 바탕화면 -
사용은 자유입니다. 여우와 드래곤을 모티브로 재제작한 바탕화면으로, 1920*1080
해상도로 제작되었습니다
